# Hero Images

This folder contains hero banner images for the carousel.

## Recommended Images:

The website currently uses placeholder images. You can replace them with actual images that show:

1. **Family support theme** - Images showing family unity and support
2. **Medical treatment support** - Images related to healthcare and recovery
3. **Community helping hands** - Images showing community support and Ukrainian solidarity

## Image Requirements:

- **Format**: JPG, PNG, or WebP
- **Size**: 1920x1080 pixels or similar 16:9 aspect ratio
- **Style**: Warm, hopeful, and supportive atmosphere
- **File names**: hero-1.jpg, hero-2.jpg, hero-3.jpg, etc.
